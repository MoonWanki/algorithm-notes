# Algorithm notes

Coding exercise with variable algorithms

## Environment

### C

gcc (GCC) 6.3.0

- Compile 
```
gcc main.c -o main.exe
```

- Run
```
./main.exe
```

### Java

Java SE (build 1.8.0_181)

- Compile 
```
javac main.java
```

- Run
```
java main
```

### Python 3

Python 3.7.1  

- Run
```
python main.py
```

### JavaScript

Node 10.14.2

- Run
```
node main.js
```
